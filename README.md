## <a name="intro"> Introdução

### <a name="oqueeh"> O que é Aprendizado de Máquina ou Machine Learning?

Basicamente o Aprendizado de Máquina (Machine Learning) é uma **subárea da Inteligência Artificial** (IA) 
que **se concentra no desenvolvimento de algoritmos e modelos que permitam a um sistema aprender a partir de dados**
e realizar tarefas específicas sem serem explicitamente programados para isso. Em outras palavras, é uma técnica 
que permite aos computadores aprenderem com dados, identificando padrões e tomando decisões com base nesses padrões.

### <a name="tiposdeaprendizado"> Tipos de Aprendizado de Máquina
São três os principais tipos de Aprendizado de Máquina, que são definidos de acordo com a forma
como os dados são processados e como o algoritmo é treinado.

#### `Aprendizado Supervisionado`

No Aprendizado Supervisionado, **o algoritmo é treinado em um conjunto de dados rotulados, onde as saídas desejadas são conhecidas**. 
O objetivo é encontrar um modelo que possa prever corretamente a saída para novos dados. O conjunto de dados rotulados é dividido 
em **dados de treinamento e dados de teste**. O algoritmo de aprendizado é treinado com os dados de treinamento e testado com os 
dados de teste obtendo assim sua precisão. O Aprendizado Supervisionado é usado para tarefas de previsão, como previsão de vendas 
ou diagnóstico médico.

Existem duas categorias principais de Aprendizado Supervisionado:

* **Classificação**: *O objetivo é prever a classe ou categoria de uma nova observação. Por exemplo, classificar um e-mail como 
spam ou não spam*.

* **Regressão**: *O objetivo é prever um valor numérico para uma nova observação. Por exemplo, prever o preço de uma casa com base 
em suas características*.

#

#### `Aprendizado Não Supervisionado`

No Aprendizado Não Supervisionado, **o algoritmo é treinado em um conjunto de dados não rotulados, onde as saídas desejadas não são 
conhecidas**. O objetivo é encontrar padrões e estruturas nos dados que possam ajudar na tomada de decisões. O conjunto de dados não 
rotulados é dividido em **dados de treinamento e dados de teste**. O algoritmo de aprendizado é treinado com os dados de treinamento e 
testado com os dados de teste, assim como o supervisionado. O Aprendizado Não-Supervisionado é usado para tarefas de agrupamento, como 
segmentação de clientes ou identificação de tópicos em textos.

Existem duas categorias principais de Aprendizado Não Supervisionado:

* **Clustering (Agrupamento)**: *O objetivo é agrupar as observações em grupos que são similares entre si, mas diferentes dos outros 
grupos.*

* **Redução de Dimensionalidade**: *O objetivo é reduzir a dimensionalidade dos dados, mantendo as informações mais importantes.*

#

#### `Aprendizado por Reforço`

E por fim, temos o Aprendizado por Reforço, em que **o algoritmo aprende através da interação com um ambiente e recebendo feedback de 
recompensa ou punição**. O objetivo é encontrar uma política que maximize a recompensa recebida. O Aprendizado por Reforço é usado em 
tarefas de tomada de decisão, como jogos de tabuleiro ou controle de robôs.

O Aprendizado por Reforço é diferente dos outros dois tipos de Aprendizado de Máquina, porque o algoritmo não recebe um conjunto de 
dados rotulados ou não rotulados no início do processo de aprendizado. Em vez disso, o algoritmo recebe feedback em tempo real à 
medida que toma decisões e interage com o ambiente.
